## üîÑ Property Transfer Workflow

The property transfer process follows these steps:

1. **Seller Initiates Transfer**
   - Seller uploads property documents and initiates transfer to buyer

2. **Broker Verification** (if broker assigned)
   - Broker verifies transaction details and authenticity

3. **Registrar Verification**
   - Registrar office verifies documents and ownership

4. **Municipal Approval**
   - Municipal corporation grants final approval

5. **Buyer Acceptance**
   - Buyer accepts transfer and uploads acceptance documents

6. **Transfer Completed** ‚úÖ
   - Property ownership transferred on blockchain

## üéØ How to Use the Improved UI

### Dashboard
- View your role badge and quick stats
- See quick action cards for common tasks
- Check your property count and wallet info

### Properties Tab
- **Search Properties**: Enter a Property ID to view any property
- **My Properties**: View all properties you own

  - Property ID and type
  - Address and area
  - Current owner
  - Link to view documents

### Transfers Tab
- **Initiate Transfer** (Sellers only):
  - Enter Property ID
  - Enter Buyer address
  - Set price
  - Upload property documents
  - Optionally add broker address

- **View Transfer Details**:
  - Enter Transfer ID
  - See complete transfer information
  - View verification status
  - Take actions based on your role

### Register Property Tab
- **For Sellers/None**: Automatically uses your wallet address as owner
- **For Registrars**: Can enter any owner's address
- Fill in property details and upload documents

- Click "‚ÑπÔ∏è Roles & Help" button in the navigation
- View your current role information
